# test_vk
**Тестовое для стажировки в VK**

Написать скрипт на Python, который будет работать в фоне и отслеживать скорость загрузки в Steam. 

**Что нужно сделать:**   
- считывать скорость загрузки
- выводить в консоль скорость загрузки каждую минуту на протяжении пяти
- учитывать статус загрузки
- выводить, какая именно игра скачивается
